# 🍕 Food Delivery Cost & Profitability Analysis

## 🧠 Project Overview
This project analyzes food delivery orders to understand cost structures and profitability. It performs data cleaning, exploratory data analysis (EDA), cost-profit calculations, and visualizations to uncover trends and business insights in food delivery metrics.

---

## 🎯 Project Goals
- Clean and preprocess raw delivery order data  
- Perform exploratory analysis to identify patterns  
- Calculate profit and cost metrics  
- Present insights that can help with pricing or operational decisions
